## 快速导航

### 项目信息
- **版本**: 3.7.8
- **许可证**: Apache License 2.0
- **语言**: Python 3.10+
- **官方网站**: https://thingsboard.io/docs/iot-gateway/

### 主要特性
- 支持15+种工业和物联网协议
- 灵活的数据转换和映射
- 本地数据持久化
- 远程配置和管理
- 自动重连机制
- 设备生命周期管理

### 支持的协议
- **工业协议**: Modbus, OPC-UA, CAN, ODBC, BACnet
- **物联网协议**: MQTT, REST, Request, FTP, Socket, SNMP, XMPP
- **智能能源**: OCPP
- **智能建筑**: BACnet, KNX, BLE
- **可扩展**: 支持自定义连接器
